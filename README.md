# regularized-logistic-regression-from-scratch
Logistic Regression implemented from scratch using NumPy with L2 regularization, gradient descent optimization, and loss visualization.

This project implements Logistic Regression with L2 Regularization from scratch using NumPy. 
The model is trained using gradient descent and optimized using log-loss with a regularization term to prevent overfitting.

## Features
- Feature normalization
- Sigmoid activation function
- Log-loss cost function
- L2 Regularization
- Gradient Descent optimization
- Loss vs Iterations plot
- Dataset visualization

## Libraries Used
- NumPy
- Matplotlib
- Scikit-learn (only for dataset generation)

## How to Run
1. Install required libraries:
   pip install numpy matplotlib scikit-learn

2. Run the script:
   python regularized_logistic_regression.py
